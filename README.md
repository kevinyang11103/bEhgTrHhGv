# 大数据经济型酒店推荐系统 python1278

## 项目概述

本项目是一款基于协同过滤算法的酒店推荐系统，旨在为用户提供经济型酒店的智能推荐服务。通过使用大数据分析技术，结合用户偏好和行为数据，为用户推荐最合适的酒店。

## 技术栈

### 后端

- **Python**: 使用Python作为主要开发语言。
- **Django**: 基于Django框架进行后端开发。

### 数据库

- **MySQL**: 使用MySQL数据库进行数据存储，版本需为5.7或更高。

### 开发工具

- **PyCharm/vs code**: 推荐使用PyCharm或vs code进行开发。

### 前端

- **Vue.js**: 使用Vue.js作为前端框架。

## 功能模块

- 用户管理
- 酒店信息管理
- 推荐算法实现
- 数据展示与分析

## 系统角色

- 用户：普通用户和酒店管理人员
- 管理员：负责系统维护和数据管理

## 运行环境

- Python版本：3.7/3.8
- 数据库：MySQL 5.7或更高版本
- 数据库工具：Navicat11（推荐）

## 部署步骤

1. 环境配置：安装Python、MySQL、Navicat11等必要软件。
2. 克隆项目代码到本地。
3. 导入数据库结构和数据。
4. 运行项目。
